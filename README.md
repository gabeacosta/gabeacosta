I build verification infrastructure for AI agent execution.

Independent proof that an agent did what it claims: signed receipts, hash-chained
audit trails, and enforcement that runs before the tool call, not after.

**Systems:** Python, distributed verification, Ed25519 provenance, MCP governance,
self-hosted inference (Ollama / MLX), n8n orchestration, Postgres / Qdrant / Redis.

Most of my work is private. [agentreceipts](https://github.com/gabeacosta/agentreceipts) is the open verifier.
